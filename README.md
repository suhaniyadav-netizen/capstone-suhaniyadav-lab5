# TripTonic

TripTonic is a simple, elegant static website built as a capstone project to showcase travel destinations and packages. It is hand-crafted with semantic HTML and CSS and includes responsive layout considerations, accessible landmarks, and a clean visual design.

## What this project is

- A static marketing/demo site for a travel service called TripTonic.
- Built with plain HTML (`index.html`) and CSS (`style.css`).
- Includes example destination cards, package listings, and a contact form (front-end only).
- Images are included at the project root (`image_1` .. `image_4`).

## Key Features

- Clean, modern layout and typography (Google Fonts used).
- Destinations section with image cards and short descriptions.
- Packages section with price and duration details.
- Contact form UI for collecting user interest (no backend included).
- Skip link and basic accessibility considerations.

## Project Structure

- `index.html` : The main page containing site content and structure.
- `style.css`  : Stylesheet for layout, colors, and typography.
- `image_1`, `image_2`, `image_3`, `image_4` : Project images used in the Destinations section.
